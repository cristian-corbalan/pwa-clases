Aplicaciones web progresivas - Ejemplo 01

Consigna

  • Solicitar el permiso de notificaciones al usuario al dar clic en el botón.
  • Tener en cuenta los siguientes puntos:
    • El usuario ya podría haber concedido el permiso anteriormente.
    • El usuario podría haber rechazado el permiso de notificaciones.
    • El navegador del usuario podría no ser compatible con la API de notificaciones.
    • El navegador del usuario podría no ser compatible con los Service Workers (Notificaciones push)
  • Enviar una notificación al usuario y reaccionar cuando este la cliquea.
