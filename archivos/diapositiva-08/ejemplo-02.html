Información

  • Este es un ejemplo que simula como se podría realizar el proceso de sincronización.
  • En este caso, el usuario puede cargar distintos libros que, según su conexión, se sincronizarán o no con el servidor (simulado)
  • También se detectará los cambiós de conexión del usuario en tiempo real para realizar la sincronización sin tener que reiniciar la página.
  • Nota: Este es un ejemplo bastante avanzado, se combinan muchas técnicas que estuvimos viendo hasta ahora en la cursada. Asi que no te preocupes si no lo entiendes a la primera ✌️😗.
