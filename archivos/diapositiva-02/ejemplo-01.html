Aplicaciones web progresivas - Ejemplo 01

Consigna

  • Imprimir la siguiente estructura dentro del ul#tareas utilizando el array tareas: 
    <li>CONTENIDO <button>X</button></li>
  • Buscar la etiqueta input#contenido y button#crear.
  • Al hacer clic en el botón, crear una nueva tarea con el contenido del input.
  • En el caso de que el input este vacío, mostrar un mensaje en el p#respuesta
  • Cada tarea se debe de poder eliminar presionando su respectivo botón "X".
