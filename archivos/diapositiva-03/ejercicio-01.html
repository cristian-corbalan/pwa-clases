Aplicaciones web progresivas - Ejercicio 01

Consigna

  • Crear una función llamada obtenerUsuario(nombre) que:
    • Reciba un nombre de usuario.
    • Obtenga la información de dicho usuario de GitHub y muestre el objeto con sus datos en la consola.
  • La URL de GitHub con la información de usuario es: https://api.github.com/users/NOMBRE.
  • Cambiar la parte de NOMBRE por el parámetro recibido.
  • Detalles a tener en cuenta:
    • Debe realizarse una petición fetch para la obtención de datos.
    • Si la petición falla o si el usuario no existiese, la función debe devolver null.
  • ➕: Mostrar la imágen del usuario recibido en el documento HTML (div#avatar).
